# VideoAcademy

## Important commands
Create a new app
```bash
./manage.py startapp <app_name>
```

Create a new migration
```bash
./manage.py makemigrations
```

Run migrations
```bash
./manage.py migrate
```

Create a superuser
```bash
./manage.py createsuperuser
```

Run the server
```bash
./manage.py runserver
```
